# Arena Tanks (diep.io-style single-player clone)

Одиночная аркада в стиле diep.io на движке Godot 4.3: полное дерево танков,
конструктор танков, 49 ботов с ИИ, физика на RigidBody2D, геймпад и
клавиатура+мышь.

## Возможности

- **Главное меню**: Играть / Настройки / Дерево танков / Ник / Конструктор танков.
- **Дерево танков**: 1 → 3 → 9 → 27 → 81 → 243 = 364 танка, тиры 1–6.
- **9 классов оружия**: normal, sniper, turret, drone, rammer, shotgun, machinegun, mine_layer, rocket.
- **Конструктор танков**: создание нового танка с нуля или редактирование любого
  существующего (тир, родитель, класс, стволы, статы, имя, размеры).
- **Физика**: RigidBody2D для танков, снарядов и фигур — реалистичная отдача и
  нокбэк на основе массы.
- **Карта**: до 40 000 фигур (4 варианта квадратов, 3 варианта треугольников,
  9 вариантов многоугольников) + 49 ботов и игрок.
- **Лидерборд**: топ-10 в реальном времени.
- **500 имён ботов** в `data/bot_names.json`.
- **ИИ ботов**: поведения фарм/охота/дуэль/бегство/блуждание на основе
  персональных черт характера, автоматическая прокачка статов и эволюция тира.
- Полная поддержка **геймпада** и **клавиатуры+мыши** с перенастройкой кнопок в
  Настройках.

## Структура проекта

```
godot-game/
  project.godot          — конфигурация проекта, инпут-мапы, автозагрузки
  autoload/               — синглтоны: GameConfig, SaveSystem, InputManager,
                             BotNames, TankDatabase, ShapeDatabase, RunState
  data/bot_names.json     — 500 имён ботов
  scripts/
    data/                 — TankDef.gd, ShapeDef.gd (Resource-классы)
    tank/                 — TankController, WeaponSystem, BotBrain, Bullet,
                             Mine, Turret, Drone
    world/                — Shape, ShapeSpawner, Game (главная игровая сцена)
    ui/                   — MainMenu, Settings, TankConstructor, TierTreeView, HUD
  scenes/                 — .tscn файлы для всех сцен выше
  .github/workflows/      — сборка Windows-exe через GitHub Actions
```

## Сборка Windows-версии

Сборка происходит автоматически на GitHub Actions при пуше в `main`
(см. `.github/workflows/build-windows.yml`). Скачать `ArenaTanks.exe`
можно во вкладке **Actions → последний workflow run → Artifacts**.

Собрать локально (если у вас установлен Godot 4.3 с шаблонами экспорта):

```
godot --headless --path godot-game --export-release "Windows Desktop" build/windows/ArenaTanks.exe
```

## Управление

| Действие          | Клавиатура/мышь | Геймпад            |
|--------------------|-----------------|---------------------|
| Движение            | WASD            | Левый стик          |
| Прицеливание/огонь  | Мышь + ЛКМ/ПКМ  | Правый стик + RT/LT |
| Пауза               | Esc             | Start               |

Все кнопки можно переназначить в меню **Настройки**.

## Известные ограничения текущей версии

- Автор писал файлы Godot вручную без доступа к редактору Godot в песочнице
  агента — сборка и визуальная проверка происходят через GitHub Actions.
  Если что-то не запускается, проверьте лог сборки в Actions.
